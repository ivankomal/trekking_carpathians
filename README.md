# Trekking Carpathians

A web project dedicated to hiking and outdoor activities in the Ukrainian Carpathians. This website aims to inspire travelers and provide essential information about mountain routes.

## Demo
You can view the live website here:  
 [https://ivankomal.github.io/trekking_carpathians/](https://ivankomal.github.io/trekking_carpathians/)

## Key Features
- **Responsive Design:** Optimized for desktops, tablets, and mobile devices.
- **Modern UI:** Clean and intuitive interface built with modern web standards.
- **Informative Content:** Sections covering popular peaks, gear checklists, and tips for beginners.

## How to Run Locally

Clone the repository:

```bash
git clone [https://github.com/ivankomal/trekking_carpathians.git](https://github.com/ivankomal/trekking_carpathians.git)
```

Open index.html in your web browser.
